Tema WordPress klasik/hibrida untuk situs resmi SMK QUEEN AL-FALAH. Gutenberg-ready, responsif, mudah diatur, dan dilengkapi plugin pendamping untuk tipe konten sekolah serta penyiapan demo sekali klik.

== Description ==

Queen Al-Falah dirancang untuk menyajikan profil sekolah, kompetensi keahlian, berita, pengumuman, agenda, prestasi, guru, ekstrakurikuler, layanan digital, galeri, mitra industri, lowongan, dan kisah alumni.

Fitur utama:

* Arsitektur classic/hybrid yang tetap nyaman digunakan melalui editor blok Gutenberg.
* Penyiapan situs demo sekali klik untuk membantu membuat struktur awal.
* Dua belas tipe konten khusus melalui Queen Al-Falah Core: Program Keahlian, Guru, Pengumuman, Agenda, Prestasi, Ekstrakurikuler, Layanan, Galeri, Mitra, Lowongan, Alumni, dan Sarana Prasarana.
* Pengaturan Customizer untuk kontak sekolah, hero, PPDB, statistik, dan media sosial.
* Tata letak responsif dan aksesibel untuk desktop, tablet, dan ponsel.
* Tidak memerlukan page builder, framework, font eksternal, atau plugin pihak ketiga.
* Mendukung logo situs, menu WordPress, gambar unggulan, komentar bertingkat, serta blok lebar.

== Installation ==

1. Unggah folder atau berkas ZIP tema melalui Tampilan > Tema > Tambah Tema.
2. Aktifkan Queen Al-Falah.
3. Unggah dan aktifkan plugin pendamping queen-alfalah-core.zip dari paket lengkap.
4. Buka Sekolah > Penyiapan Demo dan jalankan penyiapan pada situs baru atau staging.
5. Buka Tampilan > Sesuaikan untuk memeriksa hero, gambar, warna, statistik, dan media sosial.
6. Periksa data portabel di Sekolah > Pengaturan.
7. Atur Beranda sebagai halaman depan statis melalui Pengaturan > Membaca bila belum dilakukan setup.
8. Tinjau menu, simpan ulang permalink, ganti konten contoh, dan verifikasi data sebelum publikasi.

Panduan lengkap tersedia dalam README.md dan docs/CONTENT-GUIDE.md.

== Frequently Asked Questions ==

= Apakah tema memerlukan plugin atau page builder? =

Tema tidak memerlukan page builder. Halaman dan berita dasar tetap bekerja tanpa plugin, tetapi plugin Queen Al-Falah Core yang disertakan diperlukan untuk seluruh tipe konten sekolah, meta terstruktur, pengaturan portabel, dan setup demo.

= Apakah tema mendukung Gutenberg? =

Ya. Tema menggunakan pendekatan klasik/hibrida dan mendukung editor blok, gaya blok, perataan lebar, serta konten yang dibuat dengan Gutenberg.

= Apa yang dilakukan penyiapan demo sekali klik? =

Penyiapan demo pada Queen Al-Falah Core membantu membentuk struktur awal situs, seperti halaman, menu, program terverifikasi, layanan, konten draf, dan pengaturan beranda. Jalankan pada instalasi baru atau staging, buat cadangan lebih dahulu, lalu periksa hasilnya sebelum publikasi.

= Apakah data demo boleh langsung diterbitkan? =

Tidak. Data contoh berfungsi sebagai kerangka. Administrator harus memeriksa fakta sekolah, angka statistik, jadwal, foto, izin publikasi, tautan, dan data operasional lain sebelum tayang.

= Apakah formulir PPDB dan lamaran kerja sudah memproses data? =

Tema menyediakan area informasi dan tautan/CTA. Pemrosesan formulir, pembayaran, LMS, E-Rapor, atau integrasi layanan eksternal memerlukan sistem yang dipilih sekolah dan harus dikonfigurasi terpisah.

= Bagaimana menjaga privasi siswa? =

Gunakan foto yang memiliki izin, hindari menampilkan NIK, NISN, alamat pribadi, nomor telepon pribadi, data kesehatan, atau dokumen siswa. Baca docs/CONTENT-GUIDE.md sebelum mengunggah konten.
